== Description ==

`WP TXT Sitemap` creates an Text Sitemap For your website with auto ping search engines (Google and Bing). This Plugin `may` help Google quickly discover and index your website!

- The first WP Plugin that creates TXT Sitemaps.
- It does not conflict with any XML Sitemap Plugins OR any other SEO Plugins.
- It works smoothly with Rank Math & Yoast SEO Plugin.

= TXT Sitemap Content Options =

- Categories
- Posts
- Pages

= Sitemap Auto Ping Options =

- Google
- Bing

= Updating Sitemap =

The plugin auto generate `sitemap.txt` file once activated. and:

- when any new post is published.
- when settings saved.

== Installation ==

- From `Plugins` menu click `Add New` and search for `WP TXT Sitemap`.
- Click `Install Now` and then `Activate`.
- Go To `Settings` >> `Reading` for Plugin Options.

== Frequently Asked Questions ==

= How to install this plugin? =

- From `Plugins` menu click `Add New` and search for `WP TXT Sitemap`.
- Click `Install Now` and then `Activate`.
- Go To `Settings` >> `Reading` for Plugin Options.

= How Can I access the sitemap? =

After install the plugin, You will see a notice with sitemap.txt link. also you can append sitemap.txt to your site url, Example: [sitemap.txt](https://soft.kuegy.com/sitemap.txt "kuegy soft sitemap.txt example")
